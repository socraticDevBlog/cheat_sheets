##
   python - cheat sheet 
##

Virtual environnements :

   - requis : au moins python 3.4

Windows 

   a. Créer les fichiers nécessaires

      Dans PowerShell : 

         - pwd == le répertoire où vous voulez travailler

         - $python -m venv {env} (le derniere paramètre peut être nommer autrement)
            - cette commande va créer tous les fichiers nécessaires
            - par défaut, vous allez avoir un PIP propre à votre environnement virtuel

   b. Activation de l'environnement virtuel : 

            - ${env}\Scripts\activate.ps1

   c. Désactivation de l'environnement virtuel :

      - $deactivate

Linux et MacOs 

   - même processus de création

   - pour activer le VENV, /bin/activate
